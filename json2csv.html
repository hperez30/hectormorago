Convertir json a CSV

El objetivo de este script es transformar un archivo json en formato csv, de tal modo que podamos trabajar con él con algún software que maneje datos tabulares. Este código fue creado para transformar un archivo que contenia datos acerca de los dispositivos registrados en DOMjudge.
