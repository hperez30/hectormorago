Convertir XML a CSV

El objetivo de este script es transformar un archivo xml en formato csv, de tal modo que podamos trabajar con él con algún software que maneje datos tabulares. Este código fue creado para transformar un archivo que contenia datos acerca de los dispositivos registrados en Cisco CUCM. En concreto se lanzo una consulta AXL para recuperar información acerca del número de dispositivos de cada uno de los modelos en cada una de las sedes de una organización. Sin embargo, el código puede ser facilmente modificado cambiando el nombre de las etiquetas (serán las columnas en el archivo destino, en el ejemplo output.csv) sobre las que queremos trabajar.
